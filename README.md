# 🐱 猫咪食物过敏排查记录工具

一个**离线单页 HTML 工具**，帮助你结构化记录猫咪食物过敏排查全过程——从基线建立、食物排除试验、环孢素减量追踪，到复挑战定位过敏蛋白。

## 使用方法

1. 双击打开 `index.html` 即可使用，无需安装任何软件或联网。
2. 所有数据保存在浏览器本地（`localStorage`），关闭后再打开数据不会丢失。
3. **强烈建议**定期通过「数据管理」页面导出 JSON 备份。

## 排查流程简述

本工具参考兽医皮肤科标准排除法流程，分为 5 个阶段：

| 阶段 | 内容 | 建议时长 |
|------|------|----------|
| **0 基线建立** | 录入猫咪档案、既往饮食史、用药情况 | 首次使用 |
| **1 驱虫记录** | 记录并确认驱虫执行情况 | 视实际情况 |
| **2 食物排除试验** | 严格喂食水解蛋白粮，零零食零偷吃 | **8-12 周** |
| **3 复挑战** | 逐一引入单一蛋白来源，观察症状变化 | 每种 1-2 周 |
| **4 长期管理** | 确定安全/回避食物清单，长期维护 | 持续 |

### 环孢素与食物排查的关系

- 环孢素可抑制过敏症状，在食物排除试验期间需在兽医指导下逐步减量至停药。
- **只有在环孢素停药后症状仍能通过饮食控制**，才能确认食物排除有效。
- 本工具会追踪环孢素用药状态，并在趋势图中标注停药时间点，帮助对照观察。

## 功能模块

- **猫咪档案**：基本信息、症状部位、既往蛋白来源
- **驱虫记录**：驱虫日期和产品记录
- **每日记录**：抓挠频率/耳部/皮疹/呕吐/便便/食物/药物/排查严格性
- **环孢素追踪**：用药频次、减量节点、停药日期、用药日历
- **趋势分析**：抓挠频率趋势图、按周汇总、污染日标记
- **复挑战**：逐一测试蛋白来源，记录反应，定位过敏原
- **数据管理**：JSON 导出/导入、数据清空

## 数据备份

- 通过「数据管理 → 导出数据」下载 JSON 文件
- 需要恢复时通过「导入数据」选择之前的 JSON 文件
- 建议每周备份一次，重要操作前（如清空数据）务必先备份

## 技术说明

- 纯静态 HTML + CSS + JavaScript，无任何外部依赖
- 离线可用，无需联网
- 数据存储于浏览器 `localStorage`
- 趋势图使用纯 SVG 绘制
